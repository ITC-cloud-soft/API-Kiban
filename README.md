# AEON IBS components

A components library for AEON IBS Project.

## Documents

### For users

Open Storybook and see `DOCS` section.

### For contributors

See `/docs/contribute`.

## Development

### Storybook

Run `yarn storybook`

### Vanilla Example for HTML pages

Run `yarn start:vanilla-examples`

## Build

### For Distribute

Run `yarn dist`

### Storybook

Run `yarn build:storybook`

### Vanilla Example for HTML pages

Run `yarn build:vanilla-examples`
